Internal Monitoring Document

PT. Aisin Indonesia Automotive

DONNA (Document Internal Monitoring Application) is a web-based application for managing and monitoring company documents, from uploading and approval to distribution and status monitoring, to streamline administrative processes.

This project was developed using PHP with the Laravel framework, MySQL, JavaScript, Bootstrap, jQuery, and libraries such as FPDI, PhpWord, and PhpSpreadsheet to support document management and manipulation.

In addition, DONNA is also integrated with each department's WhatsApp group using the "Ruang WA" API to send broadcast notifications regarding documents that will be obsolete, so that information can be received more quickly and help departments in updating documents on time.

Tech Used

  • PHP (Laravel)
  • JQuery
  • Bootstrap Frontend
  • MySQL Database
  • Linux Server Deployment

Q-Card Management System

PT. Liugong Machinery Indonesia

Q-Card Management System is a web-based application used to manage the part claims process, integrated with Q-Card vouchers. This system facilitates the recording, validation, and monitoring of claims, as well as voucher management, making the administrative process more structured, faster, and easier to monitor.

This project was developed using ASP.NET with Razor View as the frontend templating and Kendo UI to support an interactive and responsive interface, simplifying data management and transaction processing within the system.

Furthermore, the system utilizes Hangfire as a background job scheduler to send automatic notifications regarding part claim status and Q-Card registration, ensuring real-time user information and more effective monitoring.

Tech Used

  • .Net 8 with a custom built MVC framework (ASP.NET)
  • SQL Server Database
  • Razor and Kendo UI on the Frontend
  • Hangfire as a background job scheduler
  • Windows Server Deployment
